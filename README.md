# React App Template

Basic boilerplate & scaffold to be extended for new React app intialisation

Built utilising [https://github.com/facebook/create-react-app](create-react-app)

## Get going

**Pre-requisites**

```
Node ^V8.0.0
Yarn >=V1.7
```
**Install**

```
yarn install
```

**Build**

```
#DEV
yarn start

#PRODUCTION
yarn build
```

**Test**

```
yarn test
```
